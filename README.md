# 品管考試模擬器

一個功能完整的品質管理考試模擬練習系統，提供兩種測驗模式和即時答題回饋。

## ✨ 功能特色

### 🎯 雙測驗模式

#### 📝 模擬測驗
- ⏱️ 90分鐘倒數計時
- 📊 隨機抽取100題
- ✅ 60分及格制
- 🚀 答題後自動前進
- 📈 測驗結束顯示詳細成績

#### ✏️ 一般測試
- ⏳ 不限時間練習
- 🔀 所有題目隨機排序
- ✅ 即時綠色/紅色回饋
- 🎯 答對自動前進，答錯停留學習
- 📊 即時統計顯示

### 🎨 使用者體驗
- 🏠 返回首頁按鈕（附確認保護）
- ⌨️ 鍵盤導航支援（左右箭頭）
- 📱 響應式設計
- 🎨 現代化UI設計
- ✨ 流暢動畫效果

## 🚀 快速開始

### 前置需求
- Python 3.x
- 現代瀏覽器（Chrome、Firefox、Edge等）

### 安裝步驟

1. **克隆專案**
```bash
git clone https://github.com/你的用戶名/qc-exam-simulator.git
cd qc-exam-simulator
```

2. **準備題庫資料**
```bash
# 將你的題庫 Excel 檔案放在專案根目錄
# 執行轉換腳本
python convert_data.py
```

3. **啟動本地伺服器**
```bash
python -m http.server 8000
```

4. **開啟瀏覽器**
```
http://localhost:8000/index.html
```

## 📝 題庫格式

Excel 檔案應包含以下欄位：
- 題號
- 單元
- 題目
- 選項A
- 選項B
- 選項C
- 選項D
- 答案

## 🎮 使用說明

### 模擬測驗模式
1. 點擊「模擬測驗」按鈕
2. 系統載入隨機100題並開始90分鐘倒數
3. 選擇答案後自動前進到下一題
4. 完成所有題目或時間到時點擊「提交測驗」
5. 查看詳細成績報告（分數、百分比、及格/不及格）

### 一般測試模式
1. 點擊「一般測試」按鈕
2. 系統載入所有題目並隨機排序
3. 選擇答案：
   - **答對**：顯示綠色回饋，1秒後自動下一題
   - **答錯**：顯示紅色+綠色正確答案，需手動點擊「下一題」
4. 可隨時點擊返回首頁按鈕

## 📋 主要功能實作

### Fisher-Yates 洗牌演算法
確保題目隨機且不重複

### 倒數計時器
- 精確的秒級倒數
- 視覺警告（剩餘10分鐘/5分鐘）
- 時間到自動提交

### 智慧自動前進
- 模擬測驗：任何答案後自動前進
- 一般測試：答對自動前進，答錯停留
